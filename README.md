# New SelfBot source written on [TDCli](valtman.name/telegram-cli)

# Installation

```sh
# Tested on Ubuntu 14.04

git clone https://github.com/BeatBotTeam/Self-Bot.git

cd SelfBot

chmod +x launch.sh

./launch.sh install

./launch.sh
```

* Use $help command to receive commands help message
